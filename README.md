# r2pq-cli

R2PQ CLI is the command-line interface for developers, testers, and operators working with the R2PQ suite.

It provides tools for:
- PQ key generation  
- Signing & verifying messages  
- Running PQ benchmarks  
- Interacting with r2pq-devnet  

## Structure

/commands         # CLI commands /utils            # Shared helpers /config           # Default CLI config
